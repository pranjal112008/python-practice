# 🐍 Python Practice

My Python learning journey — practice problems, coding exercises, and fundamental programming concepts implemented in Python.

A structured collection of programs and problem-solving challenges completed while learning Python through freeCodeCamp, ranging from basic syntax to algorithm-style exercises and classic DSA problems.

## 📚 About

This repository documents my Python learning progress. Programs are organized by topic, starting from basics and gradually moving into more challenging problem-solving exercises. New programs and mini-projects are added as I continue learning.
